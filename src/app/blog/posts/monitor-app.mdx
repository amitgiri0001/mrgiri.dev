## About

Monitor App is a lightweight, always-visible system monitoring tool designed to stay in the corner of your screen. It provides real-time insights into your system's performance without getting in the way of your work.

## Features

- **Always Visible**: Sticks to the corner of your screen and remains visible at all times
- **CPU Monitoring**: Real-time CPU usage tracking
- **Memory Monitoring**: RAM usage visualization
- **Network Monitoring**: Track network activity
- **Minimal Footprint**: Designed to be unobtrusive and lightweight

## Tech Stack

- **JavaScript** - 100%
- **Node.js** runtime
- Built for macOS (darwin)

## Getting Started

### Installation

```bash
nvm use 11
npm i
```

### Run

```bash
npm start
```

### Build

```bash
npm run build
```

The build will create a package according to the option provided.
